# AI Workflow Engine - Core Build (Source Code)

This is the **source code repository** for `@ai-workflow/core` (Free Build).

## For Users

If you want to **use** AI Workflow Engine, install from npm:

\`\`\`bash
npm install -g @ai-workflow/core
\`\`\`

## For Contributors

If you want to **contribute** or **build from source**:

### Installation

\`\`\`bash
git clone https://github.com/YOUR_ORG/ai-workflow-core-source.git
cd ai-workflow-core-source
npm install
npm run build
\`\`\`

### Development

\`\`\`bash
npm run dev      # Watch mode
npm test         # Run tests
npm run build    # Build
\`\`\`

## Features (Free Build)

✅ Task Management (create, update, list, status)  
✅ Workflow State Machine (6 states)  
✅ Validation System  
✅ Context Injection (AI Memory)  
✅ Role System (3-tier, 10 roles)  
✅ Git Hooks (basic)  
✅ Test Plan Generation  
✅ Rule Management  
✅ Shell Integration
